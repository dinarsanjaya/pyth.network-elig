
# PYTH NETWORK CHECK ELIG



## Deployment

To deploy this project run

```bash
  git clone https://github.com/dinarsanjaya/pyth.network-elig
```
```bash
  cd pyth.network-elig
```
```bash
  npm i
```
```bash
  isi wallet mu di "wallet.txt"
```
Untuk EVM
```bash
  node evm.js
```
Untuk solana
```bash
  node solana.js
```
